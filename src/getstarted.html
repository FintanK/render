Scaffold is a Node.js, NPM and Grunt boilerplate that focuses on automating many frontend tasks that optimize your application development, performance and user experience. It can be easily customized and built on.

Prerequisites

In order to use Scaffold you will need to have the following installed.

  • Node.js v0.10.32
  • NPM v2.1.3
  • Grunt v0.1.13
  • Bower v1.5.3

Installation

In the root directory

npm install

This project depends on a tool called ImageMagick for icon generation during the build process.

brew install ImageMagick (OSX)
apt-get install imagemagick
apt-get install graphicsmagick-imagemagick-compat

Install our dependencies using Bower

bower install (load frontend dependencies)

Building your project

With grunt installed. In the root directory run the following command.

grunt

Features

The build process performs the following tasks.

  • CSS / Javascript combination and minification.
  • SASS compiling
  • CSS vendor auto-prefixing for browser compatibility
  • Responsive images
  • Auto-generated manifest file
  • Favicon and IOS, Android and Windows tile icon generation from a single image (logo.png)

There is also a production build which adds some additional steps.

  • HTML minification

Additional

  • Watcher to reload changes made in development
  • Further compression of assets using express in gzip format
  • Favicons, IOS and Windows Tile Icons auto-generated.
  • Native notification when builds are completed.

Note: All of these features can be customize by editing Gruntfile.js

You can also run your app against Google Pagespeed using Grunt

grunt pagespeed

Running

grunt serve

This project uses port 3004 out of the box.
